# 💹 Personal Finance Dashboard

A modern and responsive **Personal Finance Dashboard** built using **HTML5, CSS3, and Vanilla JavaScript**. This application helps users monitor their financial activities through an interactive dashboard featuring transaction management, budgeting, analytics, and visual reports.

The project demonstrates advanced JavaScript concepts such as modular architecture, CRUD operations, Local Storage, state management, data visualization, and responsive dashboard design.

---

# 📌 Project Overview

Personal Finance Dashboard is a client-side web application designed to help users manage their personal finances efficiently. Users can record income and expenses, monitor monthly budgets, analyze spending habits, and visualize financial data through interactive charts.

The application stores all financial records locally using the browser's **Local Storage**, providing a seamless experience without requiring a backend database.
